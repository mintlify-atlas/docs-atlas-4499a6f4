<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Raffi up and running in minutes
  </p>
  
  <Steps>
    <Step title="Install Raffi">
      Install via your preferred package manager:
      
      ```bash Cargo
      cargo install raffi
      ```
      
      ```bash Arch Linux (AUR)
      yay -S raffi-bin
      ```
      
      ```bash Homebrew
      brew tap chmouel/raffi https://github.com/chmouel/raffi
      brew install raffi
      ```
      
      ```bash NixOS
      nix-shell -p raffi
      ```
    </Step>
    
    <Step title="Create configuration file">
      Create `~/.config/raffi/raffi.yaml` with your first launcher entry:
      
      ```yaml
      firefox:
        binary: firefox
        icon: firefox
        description: Firefox browser
      
      terminal:
        binary: kitty
        icon: kitty
        description: Terminal emulator
      ```
    </Step>
    
    <Step title="Launch Raffi">
      Run Raffi to see your configured applications:
      
      ```bash
      raffi
      ```
      
      <Accordion title="Example output">
        The launcher will display your configured entries with icons. Select an entry to launch it.
        
        For native UI mode:
        ```bash
        raffi -u native
        ```
      </Accordion>
    </Step>
    
    <Step title="Integrate with your window manager">
      Add a keybinding to launch Raffi from your window manager. For Sway:
      
      ```config
      set $menu raffi -p
      set $super Mod4
      bindsym $super+Space exec $menu | xargs swaymsg exec --
      ```
    </Step>
  </Steps>
</div>
